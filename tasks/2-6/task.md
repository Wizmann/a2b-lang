# 2-6：独一无二

## 输入

一个仅由 `a`、`b`、`c` 组成的字符串。

## 输出

将字符串按连续相同字母划分为若干段（run）。当且仅当恰好有一个 run 的长度为 1 时，输出 `true`；否则输出 `false`。

这与题面示例及给定 A=B 解法等价：例如 `aabcc` 的 run 长度为 `2,1,2`，所以输出 `true`；`abbcc` 的 run 长度为 `1,2,2`，所以输出 `true`；`babbcc` 的 run 长度为 `1,1,2,2`，所以输出 `false`。

## 限制条件

- `1 <= 输入长度 <= 7`
- 挑战目标：A=B 程序最多 13 行。

## 样例

| Input | Output |
| --- | --- |
| `aabcc` | `true` |
| `abbcc` | `true` |
| `babbcc` | `false` |
